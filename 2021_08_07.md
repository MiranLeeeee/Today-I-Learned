# DAY 6 (week1, 2021-08-07)
> ## 오늘의 정리
`트리` `트라이`
- 트리의 순회
 1. 전위순회: 현재 -> 왼쪽 -> 오른쪽
 2. 중위순회: 왼쪽 -> 현재 -> 오른쪽
 3. 후위순회: 왼쪽 -> 오른쪽 -> 현재

- 트라이
1. 트리 구조 형태로 문자열 검색에 좋은 자료구조
2. 각 노드의 자손은 공통된 접두사를 가지고 있음
3. 검색어와 같은 노드까지 이동하고 순회를 통해 자동완성 등을 구현
4. 해시테이블이나 링크드리스트로 구현가능
